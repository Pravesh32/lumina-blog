# ✍️ Lumina Blog

A sleek, multi-user blog application built entirely with **vanilla HTML, CSS, and JavaScript**. Features client-side authentication, dark mode, real-time search, likes, post editing, and persistent storage using `localStorage`.

![Preview](https://via.placeholder.com/800x400?text=Lumina+Blog+Preview) *(Replace with a screenshot of your app)*

## ✨ Features
- 🔐 **Multi-User Auth** – Register, login, and switch accounts seamlessly
- 🌙 **Dark/Light Mode** – Theme preference saved across sessions
- 🔍 **Live Search** – Filter posts by title, content, or author in real-time
- ❤️ **Like System** – Per-user toggle with live counters
- ✏️ **Edit & Delete** – Authors can modify or remove their posts
- 💾 **Client-Side Persistence** – All data survives page refreshes via `localStorage`
- 🛡️ **XSS Protection** – User input is safely escaped before rendering
- 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop
- ⏱️ **Relative Timestamps** – "2 hours ago", "Just now", etc.
